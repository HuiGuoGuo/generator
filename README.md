# generater

#### 项目介绍
快速构建项目框架及快速生成Mybatis代码

#### 软件架构
springboot , mybatis


#### 安装教程

1. 下载代码
2. 修改application.yml文件里面的数据库为想要生成Mybatis代码的数据库，并修改账号密码
3. 打开http://localhost:80
